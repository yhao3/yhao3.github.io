Firstrade 出金教學，以國泰世華銀行為例（2024 年）

October 25, 2024 · Hao

Firstrade 出金教學#

進入美股市場已經 6 個月了，在這牛市中也嚐到了不少甜頭，所以上個月第一次試著把一些獲利提領出來，畢竟網路上滿多人反映過 Firstrade 出金的流程不太順利，所以還是要實際走過一次流程才能心安。

這篇文章就來分享一下 Firstrade 出金的流程。

設定收款帳戶、填寫出金申請#

  1. 登入 Firstrade 後台

  2. 點擊左上角的 我的賬戶 > 存款/轉戶

  3. 在 從第一證券提款 區塊中，選取 匯款

  4. 若是第一次設定，下方區塊理論上會出現「在您提交匯款申請之前, 請您先提供收款銀行賬戶信息。」的提示，直接點擊 開始 後會需要進行身份驗證，驗證通過後即可開始設定收款帳戶

  5. 接著就是填寫收款帳戶資訊：

    • 因為我的收款帳戶是使用國泰的外幣帳戶，所以可以直接打開國泰網銀 App 進入外幣帳戶，點擊 帳戶資訊 查看 SWIFT CODE 等資訊
    • 以下是填寫範例：
    • 列舉所有填寫欄位：
      • 電匯類型：選擇 國際
      • 電匯至：選擇 銀行賬戶
      • 電匯金額 ($)：填寫要提領的金額
      • 國際匯款SWIFT代碼：依照您的收款銀行提供的 SWIFT CODE 填寫
      • 收款銀行名稱：CATHAY UNITED BANK （理論上填完 SWIFT CODE 會自動帶入）
      • 收款人銀行賬戶號碼（或IBAN號碼）：填寫您的收款帳戶號碼
      • 重新輸入收款人銀行賬戶號碼：再次填寫您的收款帳戶號碼
      • 收款銀行所在城市：Taipei
      • 收款銀行所在州或省：Taiwan
      • 收款銀行所在國家或地區：TAIWAN （理論上填完 SWIFT CODE 會自動帶入）
      • 收款銀行郵編：10300
      • 需要中間銀行（可選）：不勾選
      • 附言 （如適用）：不填寫
      • 電匯用途：Investment withdrawal
      • 您要保存此銀行信息以備將來使用嗎：選擇 是

  6. 輸入完畢後，點擊 預覽 進行最後確認

  7. 確認無誤後，點擊 提交 送出申請

提交後，理論上 Firstrade 就會直接從帳戶扣除提領金額與 25 元美金的手續費。

解匯#

提交出金的申請後，我是隔天中午就收到國泰的「外匯匯入匯款」通知，接著就可以使用國泰網銀 App 進行解匯了。

但記得「收款帳戶的銀行」也會收取一筆手續費，所以最終到帳金額會扣除掉這筆手續費（我這次是被扣了 12 美金）。

總結#

Firstrade 出金流程整體來說算是順利，實際跑過一次出金流程後，也解除了我之前擔心錢領不出來的疑慮。

但因為手續費有 2 筆，所以非必要的時候還是不要頻繁提領，否則隱藏成本真的挺高的。

希望這篇文章對準備出金的朋友有所幫助，也祝各位在美股市場中獲利順利！
